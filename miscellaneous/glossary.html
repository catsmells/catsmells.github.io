tech lingo can be hard to understand sometimes.

    especially with big tech and the media having so much influence over

what being said today is what it actually is and whatnot. i figure i'd make this page an

objectively correct glossary for tech terminology.


hacker

(n.) ˈhæk ər

    one who develops a clever solution to a problem with technical knowledge.


coder

(n.) ˈkoʊ dər

    one that translates written-out plans into low-level code for reference.


scripter

(n.) ˈskrɪptə

    one that uses high-level computer languages to carry out high-level instructions, rather than use the system's native utilities.
